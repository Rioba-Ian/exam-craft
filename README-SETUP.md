# ExamCraft CBC Studio

A React application for creating and managing CBC exam papers with TanStack Router.

## Features

- **Dashboard**: Overview of your exams and quick access to main features
- **Scan & Polish (Dimension 1)**: Upload handwritten exams and convert them to polished PDFs
- **Exam Builder (Dimension 2)**: Build exams from a CBC question bank with drag-and-drop
- **Question Bank**: Browse and manage scraped CBC questions from multiple sources
- **My Exams**: View and manage your saved exams

## Technology Stack

- **React 19** with TypeScript
- **Vite 7** for fast development and building
- **TanStack Router** for file-based routing
- **pnpm** for package management

## Getting Started

### Prerequisites

- Node.js 20.19+ or 22.12+ (recommended)
- pnpm 10+

### Installation

```bash
# Install dependencies
pnpm install
```

### Development

```bash
# Start development server
pnpm dev

# Open your browser to http://localhost:5173 (or the port shown in terminal)
```

### Build

```bash
# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Routes

- `/` - Dashboard
- `/polish` - Scan & Polish (Dimension 1)
- `/builder` - Exam Builder (Dimension 2)
- `/bank` - Question Bank
- `/exams` - My Exams

## Key Features

### Scan & Polish (Dimension 1)
1. Upload handwritten exam photos
2. AI analyzes and extracts structure
3. Identifies hand-drawn objects
4. Finds replacement images online
5. Exports polished PDF

### Exam Builder (Dimension 2)
1. Browse CBC question bank
2. Filter by grade, subject, and type
3. Drag questions into sections
4. Preview and export PDF

## Styling

The app uses a custom design system with:
- **Fonts**: Cormorant Garamond (display), Outfit (body), Inconsolata (monospace)
- **Color Palette**: Forest green, sienna, amber, paper tones
- **Components**: Cards, buttons, tags, inputs with consistent styling

- Mock data is used for demonstration purposes
